1. create independent string variables for rock, paper and scissors
2. create messages for winning and losing.
3. allow user to chose between rock paper and scissors
4. user should confirm its selection
5. once user's input has been confirmed, allow broswer to pick a random string (rock || paper || scissors)
6. compare between users choice and computer's random choice
7. send a message for if the player wins or loses
